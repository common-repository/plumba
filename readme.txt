== Description ==
Online poll plugin for WordPress. Add to any page or post!
Add a new set of question with answers "Plumba" and save. Then choose the saved Plumba in your HTML-editor in WordPress.
Your visitors can now make a vote at your page or post and the results are shown in realtime.

You can see the plugin in action here: [Demo](http://plumba.org/ "Demo")

== Installation ==
1. Download the latest zip file and extract the directory it contains.
2. Upload the directory to your '/wp-content/plugins/' directory.
3. Activate the plugin on the 'Plugins' menu in the WordPress Dashboard.

The plugin requires jQuery and Twitter Bootstrap and will add those if missing in your theme.
Please change this setting in wp-admin Settings / Plumba if you already have theme with Twitter Bootstrap.

== Frequently Asked Questions ==

= What to do right after activate plumba? =
In wp-admin you now have a custom post type named "Plumba". Create a new one and publish! Then assign the new Plumba in a page or post via the button in the HTML-editor.

= What is Twitter Bootstrap? =
It's just am extension to theme development. And if your theme dont use it it will be added automatically.

= What do you mean by "online poll"? =
The bars for your questions will change over time when anyone makes a vote.

= Why will I use Plumba Poll? =
Eg, in a conference to ask questions online with the participants.

= Why is this plugin developed? =
Yes, there are a lot of poll plugins for WordPress.
This plugin is written as a part of the presentation for WordCamp Stockholm 2012.
Perhaps it will be used afterwards.
